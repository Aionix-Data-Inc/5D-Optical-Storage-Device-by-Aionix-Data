# 5D Optical Storage Device Simulation by Aionix Data

## Public Notice

This repository contains a simulation of a secure, tamper-evident, and auditable 5D optical storage pipeline, developed by Aionix Data. The code demonstrates advanced security, audit, and alerting features for next-generation data storage systems. 

**Key Features:**
- AES-256 encryption, Ed25519 signatures, SHA-256 hashing
- Role-based access control and audit logging
- Tamper detection (deletion, permission, rapid mod, forbidden keywords, renaming, size anomaly)
- Automated alerting via webhook/email (default: aionixdata@gmail.com)
- Policy-driven, modular, and extensible design

**Usage:**
- For research, demonstration, and educational purposes only.
- Not for use in production or with sensitive data without further review.
